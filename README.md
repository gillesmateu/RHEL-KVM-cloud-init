# RHEL-KVM-cloud-init

Script to automaticaly create VMs with RHEL KVM Guest Image 

1. clone the depot
2. make the script executable
3. create an `images/` subdirectory
4. go to section "Virtualization Images" at https://access.redhat.com/downloads/content/rhel
5. download qcow images to the `images/` directory
6. run the script
